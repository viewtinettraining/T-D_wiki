# VC-PSP — Viewtinet Certified Presales Solutions Professional

The VC-PSP is the official certification for professionals involved in both sales and pre-sales processes. It provides the technical and commercial depth needed to qualify opportunities, present architecture solutions, and overcome client objections.

## Course Details

| Field | Value |
|---|---|
| Duration | 8 hours |
| Delivery | Online and on-site |
| Target audience | Sales and presales professionals |
| Prerequisites | Basic knowledge of IT monitoring concepts |
| Assessment | Examination Presentation (minimum 70% to pass) |
| Certification | "Viewtinet Certified Presales Solutions Professional" |

## Overview

The VC-PSP builds on the commercial awareness introduced in the [[vsp]] and adds the technical substance required for presales engagements. Participants learn to qualify leads, understand deployment scenarios, and master hardware sizing guidelines. The course culminates in real-world case studies.

## Learning Objectives

- Gain a comprehensive understanding of the full Viewtinet product portfolio and feature sets
- Learn how to identify and qualify leads based on client needs
- Understand deployment scenarios and architecture considerations across [[standalone-vs-cluster]] options
- Master sizing guidelines based on traffic volume, user base, and data retention requirements
- Analyze real-world presales and deployment cases drawn from customer projects

## Full Agenda

1. **Welcome & Introduction** — Course structure, value proposition for the Presales role
2. **Products & Features** — Viewtinet product portfolio overview, solution-to-use case mapping, [[inventory-management]] / [[autodiscovery]] / [[configuration-manager]] features
3. **Viewtilog** — Log data processing fundamentals, use cases, deployment recommendations, key differentiators; see [[viewtilog-overview]]
4. **Viewtimon** — Network traffic monitoring principles, typical deployment scenarios, architecture considerations, visibility value proposition; see [[viewtimon-overview]]
5. **Viewtify QoS** — Traffic prioritization concepts, policy management, deployment options, infrastructure requirements; see [[viewtiqos-overview]]
6. **Viewtimanager** — Centralized management features, scalable and unified operations, administrative workflows; see [[viewtimanager-overview]]
7. **Viewtisight** — Dashboard and reporting capabilities, operational/business/executive use cases, data visualization best practices; see [[viewtisight-overview]]
8. **Hardware & Sizing Guidelines** — Appliance models and technical specifications, best-fit scenarios, proper solution sizing based on traffic volume, user base, data retention
9. **Opportunity Identification** — Applying product knowledge to detect leads, matching solutions to client needs, qualification criteria
10. **Case Studies** — [[viewtimon-overview]], [[viewtilog-overview]], and ViewtifyQoS project examples from real deployments

## Assessment

Participants must pass an Examination Presentation with a minimum score of 70% to earn the "Viewtinet Certified Presales Solutions Professional" certificate.

## Position in the Certification Path

The VC-PSP is part of the Presales vertical. See [[certification-paths]] for the full map. For deeper technical work after VC-PSP, engineers should pursue [[vc-oa]] as the analytical core certification.
