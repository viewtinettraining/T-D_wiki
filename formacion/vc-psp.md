# VC-PSP — Viewtinet Certified Presales Solutions Professional

The VC-PSP is the official certification for professionals involved in both sales and pre-sales processes. It provides the technical and commercial depth needed to qualify opportunities, present architecture solutions, and overcome client objections.

## Course Details

| Field | Value |
|---|---|
| Duration | 8 hours |
| Delivery | Online and on-site |
| Target audience | Sales and presales professionals |
| Prerequisites | Basic knowledge of IT monitoring concepts |
| Assessment | Examination Presentation (minimum 70% to pass) |
| Certification | "Viewtinet Certified Presales Solutions Professional" |

## Overview

The VC-PSP builds on the commercial awareness introduced in the [Vsp](vsp.md) and adds the technical substance required for presales engagements. Participants learn to qualify leads, understand deployment scenarios, and master hardware sizing guidelines. The course culminates in real-world case studies.

## Learning Objectives

- Gain a comprehensive understanding of the full Viewtinet product portfolio and feature sets
- Learn how to identify and qualify leads based on client needs
- Understand deployment scenarios and architecture considerations across [Standalone Vs Cluster](../conceptos/standalone-vs-cluster.md) options
- Master sizing guidelines based on traffic volume, user base, and data retention requirements
- Analyze real-world presales and deployment cases drawn from customer projects

## Full Agenda

1. **Welcome & Introduction** — Course structure, value proposition for the Presales role
2. **Products & Features** — Viewtinet product portfolio overview, solution-to-use case mapping, [Inventory Management](../configuracion/inventory-management.md) / [Autodiscovery](../configuracion/autodiscovery.md) / [Configuration Manager](../configuracion/configuration-manager.md) features
3. **Viewtilog** — Log data processing fundamentals, use cases, deployment recommendations, key differentiators; see [Viewtilog Overview](../productos/viewtilog-overview.md)
4. **Viewtimon** — Network traffic monitoring principles, typical deployment scenarios, architecture considerations, visibility value proposition; see [Viewtimon Overview](../productos/viewtimon-overview.md)
5. **Viewtify QoS** — Traffic prioritization concepts, policy management, deployment options, infrastructure requirements; see [Viewtiqos Overview](../productos/viewtiqos-overview.md)
6. **Viewtimanager** — Centralized management features, scalable and unified operations, administrative workflows; see [Viewtimanager Overview](../productos/viewtimanager-overview.md)
7. **Viewtisight** — Dashboard and reporting capabilities, operational/business/executive use cases, data visualization best practices; see [Viewtisight Overview](../productos/viewtisight-overview.md)
8. **Hardware & Sizing Guidelines** — Appliance models and technical specifications, best-fit scenarios, proper solution sizing based on traffic volume, user base, data retention
9. **Opportunity Identification** — Applying product knowledge to detect leads, matching solutions to client needs, qualification criteria
10. **Case Studies** — [Viewtimon Overview](../productos/viewtimon-overview.md), [Viewtilog Overview](../productos/viewtilog-overview.md), and ViewtifyQoS project examples from real deployments

## Assessment

Participants must pass an Examination Presentation with a minimum score of 70% to earn the "Viewtinet Certified Presales Solutions Professional" certificate.

## Position in the Certification Path

The VC-PSP is part of the Presales vertical. See [Certification Paths](certification-paths.md) for the full map. For deeper technical work after VC-PSP, engineers should pursue [Vc Oa](vc-oa.md) as the analytical core certification.
